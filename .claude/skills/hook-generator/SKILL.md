# Hook Generator — 标题/钩子生成

> 详细文档: [_architecture/L2-execution/core/hook-generator/SKILL.md](_architecture/L2-execution/core/hook-generator/SKILL.md)

## 执行协议

### 1. 标题风格库

| 风格 | 模式 | 示例 |
|------|------|------|
| numeric | 数字+利益 | "5个方法让你效率翻倍" |
| suspense | 悬念+意外 | "用了这个方法后，我再也不..." |
| pain_point | 痛点+暗示 | "还在为xxx烦恼？" |
| contrast | 前后对比 | "从A到B，我只做了这一件事" |
| question | 引发好奇 | "为什么xxx总是xxx？" |
| benefit | 利益承诺 | "让你xxx的终极方法" |
| urgency | 紧迫感 | "2024年最值得xxx" |

### 2. 平台规则

| 平台 | 长度限制 | 推荐风格 | 注意事项 |
|------|---------|---------|---------|
| 小红书 | ≤20字 | 数字/痛点/悬念 | 避免绝对化用语 |
| 公众号 | ≤64字 | 疑问/对比/利益 | 避免震惊体 |
| Twitter | ≤100字 | 悬念/对比/数字 | 加 thread 标记 |

### 3. 输出格式

生成 3-5 个候选标题，每个包含：标题内容、风格类型、吸引力评分(0-100)、选择理由。
标注推荐的最佳标题及原因。

### 4. 评分维度

- 吸引力 (30%): 是否引发点击欲望
- 清晰度 (25%): 主题是否明确
- 平台适配 (20%): 是否符合平台风格
- 长度 (15%): 是否在最佳范围
- 独特性 (10%): 是否有差异化
