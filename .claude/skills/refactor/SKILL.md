# Refactor — 代码重构

> 详细文档: [_architecture/L2-execution/core/refactor/SKILL.md](_architecture/L2-execution/core/refactor/SKILL.md)

## 重构类型

| 类型 | 操作 |
|-----|------|
| 提取 | 提取方法、类、接口 |
| 内联 | 内联方法、变量 |
| 移动 | 移动方法、字段 |
| 重命名 | 改善命名 |
| 简化 | 简化条件、合并重复 |

## 重构原则

- 保持行为不变
- 小步前进
- 测试保护
