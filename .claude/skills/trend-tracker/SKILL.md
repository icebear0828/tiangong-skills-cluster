# Trend Tracker — 热点追踪

> 详细文档: [_architecture/L2-execution/extended/trend-tracker/SKILL.md](_architecture/L2-execution/extended/trend-tracker/SKILL.md)

## 执行协议

### 分析维度

- 当前热门话题
- 上升趋势关键词
- 平台特定热点
- 内容创作机会
- 借势角度建议

### 输出格式

```
热门趋势:
1. [趋势] — 热度: XX — 生命周期: 上升/顶峰/下降
2. ...

内容机会:
- 角度1: ...
- 角度2: ...

借势建议: ...
```

## 追踪任务

$ARGUMENTS

---

请分析相关领域的热点趋势，输出内容创作机会和借势建议。
