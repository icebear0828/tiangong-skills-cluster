# Content Curator — 内容策展器

> 详细文档: [_architecture/L2-execution/extended/content-curator/SKILL.md](_architecture/L2-execution/extended/content-curator/SKILL.md)

## 执行流程

1. 分析学习主题和目标
2. 搜索相关资源（文档、教程、论文、代码示例）
3. 按质量和相关性评分排序
4. 推荐最优学习路径

## 输出: 排序资源列表 + 学习路径建议
