# 爆款模式库

## 概述

本文档汇总了经过验证的爆款内容模式，供 hook-generator、narrative-builder 和 virality-scorer 参考使用。

## 标题公式库

|------|------|------|
| 具体数字 | "X个方法/技巧/步骤..." | "7个让你早起不痛苦的方法" |

| 时间框架 | "从X到Y，只用了Z天" | "从0到10万粉，只用了3个月" |
| 清单 | "盘点：X个值得收藏的..." | "盘点：8个超实用的效率工具" |

### 悬念型公式

| 公式 | 模板 | 示例 |

| 公式 | 模板 | 示例 |

## 开头钩子模式

### 故事开头

```
模式: 场景描述 → 冲突/问题 → 转折暗示

示例:
"上周，我差点辞职。
连续加班一个月后，我躺在床上问自己：这真的是我想要的生活吗？
直到我看到一句话，改变了我的想法..."
```

### 数据开头

```
模式: 惊人数据 → 关联读者 → 解决方案暗示

示例:
"一项调查显示，75%的人每天花在手机上的时间超过4小时。
如果你也是其中之一，这篇文章可能会帮到你。"
```

### 共鸣开头

```
模式: 描述共同经历 → 点出痛点 → 给予希望

示例:
"你有没有过这种感觉：明明很累，却怎么也睡不着？
躺在床上，脑子里全是明天的工作，越想睡越清醒。
别担心，你不是一个人，而且这个问题是可以解决的。"
```

### 反常识开头

```
模式: 抛出反常识观点 → 制造认知冲突 → 吸引继续阅读

示例:
"努力，其实是一种懒惰。
听起来矛盾？但这正是大多数人进步缓慢的真正原因。
让我解释..."
```

### 提问开头

```
模式: 抛出问题 → 引发思考 → 承诺答案

示例:
"为什么有些人看起来每天都很轻松，却能把事情都做好？
而你忙得团团转，却总觉得时间不够用？
今天分享的这个方法，可能会颠覆你的认知。"
```

## 叙事结构模式

### AIDA 结构

```
A - Attention (注意): 吸引眼球的开头
I - Interest (兴趣): 维持兴趣的内容展开
D - Desire (欲望): 激发想要的欲望
A - Action (行动): 引导行动的结尾

适用: 营销类、转化类内容
```

### PAS 结构

```
P - Problem (问题): 描述读者面临的问题
A - Agitation (煽动): 加剧问题的严重性
S - Solution (解决): 提供解决方案

适用: 痛点类、解决方案类内容
```

### 故事弧结构

```
1. 设置 (Setup): 介绍背景和人物
2. 冲突 (Conflict): 遇到问题或挑战
3. 上升 (Rising): 尝试解决、情节发展
4. 高潮 (Climax): 关键转折点
5. 下降 (Falling): 问题解决过程
6. 结局 (Resolution): 结论和启示

适用: 故事类、经历分享类内容
```

### 倒金字塔结构

```
最重要 → 次重要 → 补充信息

1. 核心观点/结论
2. 支撑论据
3. 背景信息
4. 补充细节

适用: 快速消费内容、新闻类
```

## 情感触发模式

### 高传播情感

| 情感 | 触发方式 | 示例 |
|------|---------|------|
| 惊讶 | 反常识、意外结果 | "原来这才是真相" |
| 共鸣 | 描述共同经历 | "你是不是也有过这种感觉" |
| 焦虑 | 点出潜在风险 | "不改变就会落后" |
| 希望 | 提供解决方案 | "其实可以很简单" |
| 愤怒 | 揭露不公/问题 | "这太不合理了" |
| 感动 | 真实故事、温情 | "谢谢你曾经的坚持" |

### 情感配比建议

```
内容类型    | 主情感     | 辅情感
-----------|----------|--------
教程类      | 希望      | 共鸣
故事类      | 感动      | 惊讶
观点类      | 共鸣      | 愤怒
种草类      | 期待      | 信任
```

## 社交货币模式

### 分享动机

| 动机 | 描述 | 设计方式 |
|------|------|---------|
| 自我表达 | 分享代表"我是谁" | 加入身份标签、观点金句 |
| 社交互惠 | 分享有价值信息给朋友 | 提供实用资源、工具 |
| 信息传递 | 传播重要/新鲜信息 | 增加新闻性、时效性 |
| 情感宣泄 | 表达共鸣或立场 | 引发共鸣、情感释放 |
| 自我形象 | 展示品味/能力 | 提供"高级"内容 |

### 金句设计原则

```
1. 简短有力 (10-20字)
2. 观点鲜明
3. 朗朗上口
4. 适合脱离上下文单独理解
5. 适合截图分享

示例:
- "努力是一种懒惰的选择"
- "你的时间用在哪里，你的人生就在哪里"
- "不是因为看到希望才坚持，而是坚持才能看到希望"
```

## 平台爆款特征

### 小红书爆款要素

```
视觉: 首图吸睛 + 图文并茂
标题: emoji + 口语化 + 悬念/价值
开头: 前3行抓住注意力
内容: 干货 + 情感 + 互动引导
话题: 热门话题 + 精准话题组合
```

### 公众号爆款要素

```
标题: 金句感 + 明确价值
开篇: 故事/观点直接切入
结构: 小标题分段 + 逻辑递进
金句: 可截图分享的句子
结尾: 情感升华 + 引导互动
```

### Twitter 爆款要素

```
首条: 极强钩子，必须吸引展开
观点: 鲜明有力，引发讨论
结构: 每条独立 + 整体连贯
结尾: 明确CTA + 转发邀请
互动: 引发评论和引用
```

## 避坑指南

### 常见失败模式

| 模式 | 问题 | 改进 |
|------|------|------|
| 标题党 | 内容与标题不符 | 确保标题与内容一致 |
| 堆砌干货 | 信息密度过高 | 适度留白，加入情感 |
| 自嗨写作 | 只从作者角度 | 始终从读者角度出发 |
| 结构混乱 | 逻辑不清晰 | 先列大纲，再填充 |
| 无互动设计 | 缺少互动引导 | 加入问题、投票、讨论 |

### 敏感区域

```
避免:
- 过度承诺 (保证、一定)
- 绝对化表达 (最、唯一)
- 敏感话题 (政治、宗教等)
- 虚假信息
- 侵权内容
```
