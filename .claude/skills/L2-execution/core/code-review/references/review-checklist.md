# 代码审查检查清单

## 安全检查

### 通用
- [ ] 无硬编码密码/密钥
- [ ] 敏感数据已加密
- [ ] 输入已验证和清洗
- [ ] 输出已编码
- [ ] 使用安全的随机数生成器

### Web 应用
- [ ] 无 SQL 注入风险
- [ ] 无 XSS 漏洞
- [ ] 无 CSRF 漏洞
- [ ] 正确设置 CORS
- [ ] 使用 HTTPS
- [ ] 安全的 Cookie 设置

### 认证授权
- [ ] 正确验证用户身份
- [ ] 检查授权后才执行操作
- [ ] 密码使用强哈希（bcrypt/argon2）
- [ ] 会话管理安全

## 正确性检查

### 逻辑
- [ ] 条件判断正确
- [ ] 循环边界正确
- [ ] 递归有终止条件
- [ ] 算法复杂度合理

### 边界条件
- [ ] 空值/null 处理
- [ ] 空集合处理
- [ ] 数值溢出检查
- [ ] 字符串边界

### 错误处理
- [ ] 异常正确捕获
- [ ] 错误信息有意义
- [ ] 资源正确释放
- [ ] 事务正确回滚

## 性能检查

### 数据库
- [ ] 无 N+1 查询
- [ ] 使用索引
- [ ] 批量操作
- [ ] 连接池使用

### 内存
- [ ] 无内存泄漏
- [ ] 大对象及时释放
- [ ] 避免不必要的复制

### 算法
- [ ] 时间复杂度合理
- [ ] 空间复杂度合理
- [ ] 使用合适的数据结构

## 可维护性检查

### 代码结构
- [ ] 函数长度 < 50 行
- [ ] 类职责单一
- [ ] 模块边界清晰
- [ ] 依赖关系简单

### 命名
- [ ] 变量名有意义
- [ ] 函数名表达行为
- [ ] 无魔法数字
- [ ] 命名风格一致

### 注释
- [ ] 复杂逻辑有注释
- [ ] 公共 API 有文档
- [ ] 注释与代码一致
- [ ] 无过时注释

### 重复
- [ ] 无重复代码块
- [ ] 公共逻辑已提取
- [ ] 配置已外部化

## 风格检查

### 格式
- [ ] 缩进一致
- [ ] 行长度合理
- [ ] 空白使用一致
- [ ] 括号风格一致

### 导入
- [ ] 无未使用的导入
- [ ] 导入顺序正确
- [ ] 无循环导入

### 文件组织
- [ ] 文件长度合理
- [ ] 相关代码在一起
- [ ] 测试与代码分离

## 问题模板

```markdown
### Issue: [简短描述]

**严重级别**: critical / major / minor / suggestion
**类别**: security / bug / performance / style / maintainability
**位置**: 文件名:行号

**代码片段**:
```language
问题代码
```

**问题描述**:
详细说明问题...

**建议修复**:
```language
修复后的代码
```
```
