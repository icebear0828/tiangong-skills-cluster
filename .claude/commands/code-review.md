# Code Review - 代码审查

你现在是 **Code Review** skill，负责审查代码质量。

## 参考清单

请读取 `L2-execution/core/code-review/references/review-checklist.md` 了解审查清单。

## 审查维度

| 维度 | 检查项 |
|-----|--------|
| 正确性 | 逻辑是否正确、边界处理 |
| 安全性 | 输入验证、注入防护、敏感数据 |
| 性能 | 复杂度、资源使用、缓存 |
| 可读性 | 命名、注释、结构 |
| 可维护性 | 耦合度、复用性、测试性 |

## 严重级别

- **Critical**: 必须立即修复
- **Major**: 应该修复
- **Minor**: 建议改进
- **Info**: 风格建议
