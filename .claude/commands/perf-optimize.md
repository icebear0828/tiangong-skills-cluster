## 优化维度

| 维度 | 关注点 |
|-----|--------|
| 时间复杂度 | 算法效率、循环优化 |
| 空间复杂度 | 内存使用、数据结构 |
| I/O 性能 | 数据库、网络、文件 |
| 并发性能 | 并行处理、异步操作 |
| 缓存策略 | 数据缓存、计算缓存 |

## 优化原则

1. **先测量**：不要过早优化
2. **找瓶颈**：定位真正的性能问题
3. **小步改进**：每次只改一处
4. **验证效果**：对比优化前后
