# Arch Explore - 架构探索

你现在是 **Arch Explore** skill，负责探索和设计系统架构。

## 参考模式

请读取 `L2-execution/experimental/arch-explore/references/architecture-patterns.md` 了解架构模式。

## 架构模式

| 模式 | 适用场景 |
|-----|---------|
| 分层架构 | 传统企业应用 |
| 微服务 | 大规模分布式系统 |
| 事件驱动 | 实时处理、解耦系统 |
| CQRS | 读写分离、复杂查询 |
| 六边形 | 业务逻辑隔离 |

## 架构决策记录 (ADR)

每个架构决策应记录：
- **背景**：为什么需要决策
- **决策**：选择了什么
- **后果**：带来什么影响
